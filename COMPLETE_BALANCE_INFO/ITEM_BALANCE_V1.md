/================================================================================
[ ITEM STAT DEFINITION & MECHANICS ]
================================================================================/

1. 기본 생존 및 자원 (Basic Resources)
- maxHp        : 최대 체력
- maxMp        : 최대 마나
- maxStamina   : 최대 스테미나
- hpRegen      : 초당/턴당 체력 회복량
- mpRegen      : 초당/턴당 마나 회복량
- maxTurn      : [RARE] 전투 시 사용 가능한 총 턴 수 (특수 자원)

2. 공격 및 속성 (Offense & Elements)
- meleeAtk     : 물리 타입 스킬의 베이스 공격력
- magicAtk     : 마법 타입 스킬의 베이스 공격력
- {Ele}Atk     : 해당 속성(fire, ice 등)의 베이스 공격력
- {Ele}Res     : 해당 속성 데미지 % 감소 (합산 적용)
- {Ele}Pen     : 해당 속성 저항 % 관통 (합산 적용)
- [Elements]   : phys, fire, ice, lightning, earth, holy, dark, chaos, toxic

3. 전투 보조 (Combat Utility)
- critRate     : 치명타 확률
- critDmg      : 치명타 데미지 배율
- accuracy     : 명중률
- dodge        : 회피율

4. 상태이상 (Status Effects)
- statusResist : 모든 상태이상에 대한 공통 저항
- {Status}Res  : 개별 상태이상 저항 (5종류: bleed, poison 등)
- {Status}Pen  : 개별 상태이상 관통 확률 (5종류)

--------------------------------------------------------------------------------
[ 공식 메커니즘 (Formulas) ]

A. 스킬 데미지 공식 (Skill Damage)
* 조건: 스킬은 [Melee/Magic] 중 택 1 + [1개 속성] 고정
* 예시 (물리 타입 + 화염 속성 스킬 기준):
  Damage = ((meleeAtk + fireAtk) * Scaling - Target_physDef)
  * (1 - (Target_Res(phys+fire) - Attacker_Pen(phys+fire)))
* 특이사항: Pen > Res 일 경우 (1 - (음수)) 가 되어 데미지가 증폭됨 (예: 결과값이 -0.3이면 1.3배 적용)

B. 상태이상 발생 공식 (Status Chance)
* 조건: 즉발 데미지 기술에 부수적으로 발동
* 공식: (Base_Chance + {Status}Pen - Target_{Status}Res) %

--------------------------------------------------------------------------------
[ 슬롯별 할당 능력치 (Equipment Slots) ]

HEAD (투구)
- Main: maxHp, maxMp (혹은 둘 다), {Ele}Res
- Sub : statusResist, {Status}Res, accuracy

BODY (갑옷)
- Main: maxHp, maxMp (혹은 둘 다), {Ele}Res
- Sub : hpRegen, mpRegen, dodge

BOOTS (신발)
- Main: maxStamina, {Ele}Res
- Sub : dodge, {Status}Res, hpRegen

GLOVES (장갑)
- Main: {Ele}Res, {Ele}Pen
- Sub : critRate, accuracy, meleeAtk, magicAtk

NECKLACE (목걸이)
- Main: {Ele}Atk, {Ele}Pen, critDmg
- Sub : maxMp, mpRegen, statusResist

RING (반지)
- Main: meleeAtk, magicAtk, critRate
- Sub : {Ele}Atk, accuracy, maxStamina

ARTIFACT (아티팩트)
- Main: {Ele}Pen, critDmg
- Sub : 모든 능력치 중 무작위 변동폭 높음

### [ 빌드별 밸런스 분석 결과 ]

#### 1. "중갑 + 1H + 방패" vs "중갑 + 양손"
* **생존력 차이**: 전설 등급에서 방패 조합은 **HP 2,500**이라는 경이로운 수치를 기록합니다. 양손 조합보다 HP가 450이나 높고 추가 방어력(100)까지 있어, 보스의 강력한 물리 스킬을 정면에서 받아내기에 최적화되어 있습니다.
* **공격력 차이**: 양손 전사는 ATK가 250 더 높고 **CritDmg 120%** 보너스가 있어, 실제 화력은 방패 전사의 2배 이상입니다.

#### 2. "경갑 + 1H + 보조무기"의 유니크함
* 이 빌드는 전설 등급에서 **physPen 52%**를 달성합니다.
* 이는 중갑 전사(40%)보다 높은 수치로, **"단단한 적을 더 잘 잡는 기민한 자"**라는 컨셉이 명확합니다. 또한 보조 무기의 명중률(+20%)과 장갑/신발의 명중률이 더해져 회피가 높은 적에게 절대적인 우위를 점합니다.

#### 3. "경갑 + 양손" 빌드의 리스크와 리턴
* **리스크**: 전설 등급에서도 HP가 1,300에 불과해 중갑 전사의 절반 수준입니다.
* **리턴**: 하지만 **치명타 확률 30%**와 **치명타 피해 +320%** (세트 200% + 무기 120%) 조합은 적의 방어력 계산 전 데미지를 폭발시켜 전투를 순식간에 끝낼 수 있습니다.

#### 4. "경갑 + 1H + 방패" 빌드의 의외성
* 경갑 세트에 방패를 섞으면 **HP 1,750**과 **회피 31%**를 동시에 챙길 수 있습니다.
* 완전한 탱커는 아니지만, 적의 공격을 피하면서 맞아도 아프지 않은 '회피형 딜탱'이라는 독특한 포지셔닝이 가능합니다.
